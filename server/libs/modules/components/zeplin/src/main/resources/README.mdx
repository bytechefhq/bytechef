### How to find Project ID

1. Open Zeplin in your web browser (https://app.zeplin.io).
2. Navigate to your project.
3. Look at the URL in the address bar. The Project ID is the long string of characters at the end of the URL. Example: https://app.zeplin.io/project/5ab33a7928b7751764c16aaa1. The ID is 5ab33a7928b7751764c16aaa1.
