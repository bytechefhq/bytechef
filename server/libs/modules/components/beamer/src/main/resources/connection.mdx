## Connection Setup

### Find API Key

1. Navigate to your Beamer dashboard.
2. Click on **Settings**.
3. Click on **API**.
4. Click on **Create new API key**.
5. Enter name of your new API key.
6. Enable everything and **Copy** your API key.
