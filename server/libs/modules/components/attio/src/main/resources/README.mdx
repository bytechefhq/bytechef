## Connection Setup

### Find Access Token

1. Navigate to your dashboard.
2. Click on **Workspace** settings.
3. Click on **Developers**.
4. Click on **Create access token**.
5. Enable **Records**.
6. Enable **Tasks**.
7. Enable **Webhooks**.
8. Click on **Save changes**.
9. Click here to copy the access token.
