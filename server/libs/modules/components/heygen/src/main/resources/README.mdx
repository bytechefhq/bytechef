## Connection Setup

1. Navigate to your dashboard.
2. Go to your account and click on **Settings**.
3. Click on **Usage & Billing**.
4. Click on **HeyGen API**.
5. Under API Token, click **Activate**.
6. Copy your API key and use it in ByteChef.
