## Connection Setup

1. Go to your Browser Use Dashboard.
2. On the left bar, you will find **Settings** -> **API Keys**.
3. Click on **Create API Key**.
4. Add label if you want and click **Create**.
5. Copy your API Key and use it in ByteChef.
