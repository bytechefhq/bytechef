## Connection Setup

### Create API Token

1. Go to your Zendesk dashboard.
2. Click this icon.
3. Click on **Apps and integrations**.
4. Click on **API configuration**.
5. Enable API token access.
6. Click on **Save**.
7. Click on **API tokens**.
8. Click on **Add API token**.
9. Enter description of the token.
10. Click on **Save**.
11. Click on **Copy**.
12. Click on **Save**.
