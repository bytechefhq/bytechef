## Connection Setup

1. Navigate to [Claude Dashboard](https://platform.claude.com/dashboard).
2. Click on "Get API key"
3. Click on "Create key"
4. Chose workspace you want to create API key for
5. Enter name of your API key
6. Click on "Add"
7. Copy your API key
8. Click on "Close"
9. Done 🚀
