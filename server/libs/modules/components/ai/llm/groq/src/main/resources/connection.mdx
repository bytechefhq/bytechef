## Connection Setup

1. Navigate to your [Groq Console](https://console.groq.com/home).
2. Click on **API Keys**.
3. Click on **Create API Key**.
4. Enter the name of your API key and click on **Submit**.
5. Click on **Copy**.
6. Click on **Done**.
7. Done 🚀.
