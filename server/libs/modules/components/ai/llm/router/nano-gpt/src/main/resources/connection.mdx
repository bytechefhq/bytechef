## Connection Setup

1. Navigate to [NanoGPT console](https://nano-gpt.com/conversation/new).
2. Click on **API**.
3. Click on **Create API Key**.
4. Enter name of your new key.
5. Click on **Create**.
6. Here is your new API key.
7. Done 🚀.
