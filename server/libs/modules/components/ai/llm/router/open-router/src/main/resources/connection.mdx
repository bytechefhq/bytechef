## Connection Setup

1. Login into your [OpenRouter AI console](https://openrouter.ai/).
2. Click on **Get API Key**.
3. Click on **New Key**.
4. Enter name of your API key.
5. Click on **Create**.
6. Here you can copy your API Key.
7. Done 🚀.
