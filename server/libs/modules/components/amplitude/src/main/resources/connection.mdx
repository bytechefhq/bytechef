## Connection Setup

### Find API Key

1. Navigate to your dashboard.
2. Click on **Settings**.
3. Click on **Organization settings**.
4. Click on **Projects**.
5. Select project you wish to connect to.
6. Click on **Show**.
7. Here you can see API key.
