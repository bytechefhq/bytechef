# Discord Community Announcement

🚀 **ByteChef v0.25 is here!**

Hey everyone — we're thrilled to announce the launch of ByteChef!

Built for teams that value ZERO — ZERO blind trust in AI, ZERO vendor lock-in, ZERO bill shock.

Here's what's included in this release:

🔧 **Visual workflow editor** — drag-and-drop with a free-form canvas, or write workflows in code (JS, Python, Ruby, Bash)

🔌 **180+ integrations** — Slack, Jira, Salesforce, HubSpot, Google Workspace, Microsoft 365, PostgreSQL, Shopify, Stripe, and many more

🤖 **AI with oversight** — 13+ LLM providers (OpenAI, Claude, Gemini, Mistral, Groq, Bedrock) as first-class workflow components. Bring your own LLM. MCP support in both directions. Every AI call traceable.

🏠 **Self-host it** — Docker, Kubernetes with Helm charts. Open-core: platform core and basic features are Apache 2.0; advanced Enterprise features (SSO, RBAC, multi-tenancy, audit logging) are under a commercial EE license.

**30+ new components** in this release including Google Workspace suite, Microsoft 365 suite, Telegram, Trello, Monday, Jenkins, DocuSign, and more.

📋 **Full changelog:** https://github.com/bytechefhq/bytechef/releases/tag/v0.25
