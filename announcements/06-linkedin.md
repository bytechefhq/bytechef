We just launched ByteChef — and we couldn't be more excited to share it with you.

ByteChef is an open-source data orchestration platform, built for teams that want to treat automation as infrastructure — not just a collection of app connections.

Igor, Matija, and I have collectively 30+ years building integration and workflow infrastructure. We built ByteChef because we kept hitting the same walls with existing automation tools: pricing tiers that punish you for growing, vendor lock-in that makes migration painful, opaque workflows that fail silently, and platforms that force you to pick between accessible-for-business-users OR powerful-for-engineers. We wanted both.

We built ByteChef for teams that value ZERO:

→ ZERO blind trust in AI — bring your own LLM, every AI call traceable in execution history, full control
→ ZERO vendor lock-in — open-core with a real Apache 2.0 core, self-hostable anywhere
→ ZERO bill shock — consistent per-task pricing across packages, component testing included

Self-host it on your own infrastructure, with AI built in from day one. Open-core — platform core and basic features are Apache 2.0; advanced Enterprise features (SSO/SAML, RBAC, multi-tenancy, audit logging) are available in the commercial Enterprise Edition.

What's in the box:

→ Visual workflow editor with drag-and-drop — build automation without code, or write it in JavaScript, Python, Ruby, Bash. Your choice.
→ 180+ pre-built integrations — CRM, project management, communication, databases, e-commerce, Google Workspace, Microsoft 365, and more.
→ AI as a first-class workflow component — plug LLMs from 13+ providers directly into workflows. MCP support (consume tools and build workflows via MCP).

180+ integrations. 3 co-founders who believe automation should be infrastructure you own, not a subscription you rent.

And we're just getting started. Here's what's in active development:

Shipping next: AI Agents (autonomous task execution with tool use, memory, and AI guardrails), streaming chat workflows, human-in-the-loop approvals, and a Template library for sharing and importing community-built workflows.

After that: AI Copilot (natural language workflow building), Data Tables, Knowledge Base with RAG, and an MCP Server to expose workflows as tools for other AI systems.

Later: an AI Gateway with built-in observability (LLM tracing, prompt management, evaluations, alerting, rate limiting), Embedded iPaaS for SaaS companies (part of the commercial EE, early alpha), and an n8n workflow converter — plus more on our public roadmap.

If you've ever been frustrated by the cost, lock-in, or limitations of existing automation tools — give ByteChef a look.

Focus on what matters. Automate the rest.
