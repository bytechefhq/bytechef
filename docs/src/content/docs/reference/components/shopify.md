### Delete an order
Deletes an order. Orders that interact with an online gateway can't be deleted.

#### Properties

|      Name      |     Type     |     Control Type     |
|:--------------:|:------------:|:--------------------:|
| Order Id | INTEGER | SELECT  |




### Cancel an order
Cancels an order. Orders that are paid and have fulfillments can't be canceled.
