### Create channel

#### Properties

|      Name      |     Type     |     Control Type     |     Description     |
|:--------------:|:------------:|:--------------------:|:-------------------:|

| Channel name | STRING | TEXT  |  |
| Description | STRING | TEXT  |  Description for the channel.  |
