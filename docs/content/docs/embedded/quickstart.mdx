ByteChef Embedded allows you to integrate workflow automation directly into your applications, giving your users access to powerful automation capabilities without leaving your platform.

## Getting Started

1. **Set up your ByteChef account**: Create an account and configure your workspace
2. **Generate API keys**: Create secure API credentials for your application
3. **Install the SDK**: Add ByteChef to your project using npm or yarn
4. **Create workflows**: Build automation workflows that your users can access
5. **Integrate**: Embed workflow interfaces into your application

## Example Integration

```javascript
import { ByteChefClient } from '@bytechef/embedded';

const client = new ByteChefClient({
  apiKey: 'your-api-key',
  workspace: 'your-workspace'
});

// Embed a workflow in your app
client.embedWorkflow('workflow-id', '#workflow-container');
```
