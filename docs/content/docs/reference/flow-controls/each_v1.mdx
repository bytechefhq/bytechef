The Each flow control applies a function to each item in a list, in parallel. Since it processes items concurrently, there is no guarantee that items will complete in order.


**Type**: `each/v1`


## Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| items | List of items | ARRAY <details> <summary> Items </summary> [] </details> | List of items to iterate over. | false |
