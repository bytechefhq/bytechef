## Actions


### List Sites
Name: listSites

`Lists the user's Search Console sites.`

#### Example JSON Structure
```json
{
  "label" : "List Sites",
  "name" : "listSites",
  "type" : "googleSearchConsole/v1/listSites"
}
```

#### Output



Type: OBJECT


#### Properties

|     Name     |     Type     |     Description     |
|:------------:|:------------:|:-------------------:|
| siteEntry | ARRAY <details> <summary> Items </summary> [&#123;STRING\(permissionLevel), STRING\(siteUrl)&#125;] </details> | Contains permission level information about a Search Console site. For more information, see [Permissions in Search Console](https://support.google.com/webmasters/answer/2451999). |




#### Output Example
```json
{
  "siteEntry" : [ {
    "permissionLevel" : "",
    "siteUrl" : ""
  } ]
}
```

### Search Analytics
Name: searchAnalytics

`Query your data with filters and parameters that you define.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| siteUrl | Site URL | STRING | The site's URL, including protocol. For example: `http://www.example.com/`. | true |
| startDate | Start Date | DATE | Start date of the requested date range. This value is included in the range. | true |
| endDate | End Date | DATE | End date of the requested date range. This value is included in the range. | true |
| dimensions | Dimensions | ARRAY <details> <summary> Items </summary> [STRING] </details> | Dimensions to group results by. Dimensions are the group-by values in the Search Analytics page. | false |
| type | Type | STRING <details> <summary> Options </summary> <span>WEB</span>, <span>IMAGE</span>, <span>VIDEO</span>, <span>NEWS</span>, <span>DISCOVER</span>, <span>GOOGLE_NEWS</span> </details> | Filter results to the following type. | false |
| dimensionFilterGroups | Filters | ARRAY <details> <summary> Items </summary> [&#123;[&#123;STRING\(dimension), STRING\(operator), STRING\(expression)&#125;]\(filters), STRING\(groupType)&#125;] </details> | Filters to apply to the dimension grouping values. | false |
| searchType | Search Type | STRING <details> <summary> Options </summary> <span>WEB</span>, <span>IMAGE</span>, <span>VIDEO</span>, <span>NEWS</span>, <span>DISCOVER</span>, <span>GOOGLE_NEWS</span> </details> | The search type to filter for. | false |
| aggregationType | Aggregation Type | STRING <details> <summary> Options </summary> <span>AUTO</span>, <span>BY_PROPERTY</span>, <span>BY_PAGE</span> </details> | How data is aggregated. | false |
| rowLimit | Row Limit | INTEGER | The maximum number of rows to return. | false |

#### Example JSON Structure
```json
{
  "label" : "Search Analytics",
  "name" : "searchAnalytics",
  "parameters" : {
    "siteUrl" : "",
    "startDate" : "2021-01-01",
    "endDate" : "2021-01-01",
    "dimensions" : [ "" ],
    "type" : "",
    "dimensionFilterGroups" : [ {
      "filters" : [ {
        "dimension" : "",
        "operator" : "",
        "expression" : ""
      } ],
      "groupType" : ""
    } ],
    "searchType" : "",
    "aggregationType" : "",
    "rowLimit" : 1
  },
  "type" : "googleSearchConsole/v1/searchAnalytics"
}
```

#### Output



Type: OBJECT


#### Properties

|     Name     |     Type     |     Description     |
|:------------:|:------------:|:-------------------:|
| responseAggregationType | STRING <details> <summary> Options </summary> <span>AUTO</span>, <span>BY_PROPERTY</span>, <span>BY_PAGE</span> </details> | How the results were aggregated. |
| rows | ARRAY <details> <summary> Items </summary> [&#123;NUMBER\(clicks), NUMBER\(ctr), NUMBER\(impressions), [STRING]\(keys), NUMBER\(position)&#125;] </details> | A list of rows grouped by the key values in the order given in the query. |




#### Output Example
```json
{
  "responseAggregationType" : "",
  "rows" : [ {
    "clicks" : 0.0,
    "ctr" : 0.0,
    "impressions" : 0.0,
    "keys" : [ "" ],
    "position" : 0.0
  } ]
}
```
