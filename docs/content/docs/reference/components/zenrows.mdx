## Actions


### Scrape URL
Name: scrapeUrl

`Extracts HTML data from a given URL.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| url | URL | STRING | URL of the site that will be scraped. | true |
| original_status | Original Status | BOOLEAN <details> <summary> Options </summary> true, false </details> | Return the original HTTP status code from the target page. Useful for debugging in case of errors. | false |
| js_render | JS Render | BOOLEAN <details> <summary> Options </summary> true, false </details> | Enable JavaScript rendering with a headless browser. Essential for modern web apps, SPAs, and sites with dynamic content. | false |

#### Example JSON Structure
```json
{
  "label" : "Scrape URL",
  "name" : "scrapeUrl",
  "parameters" : {
    "url" : "",
    "original_status" : false,
    "js_render" : false
  },
  "type" : "zenrows/v1/scrapeUrl"
}
```

#### Output



Type: STRING

### Scrape URL With CSS Selectors
Name: scrapeUrlWithCssSelectors

`Extracts specific data from a given URL.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| url | URL | STRING | URL of the site that will be scraped. | true |
| css_extractor | CSS Extractor | ARRAY <details> <summary> Items </summary> [&#123;STRING\(key), STRING\(value)&#125;] </details> | Key-value pairs that will be scraped, where key is arbitrary parameter name and value is CSS element that you want to scrape. | true |

#### Example JSON Structure
```json
{
  "label" : "Scrape URL With CSS Selectors",
  "name" : "scrapeUrlWithCssSelectors",
  "parameters" : {
    "url" : "",
    "css_extractor" : [ {
      "key" : "",
      "value" : ""
    } ]
  },
  "type" : "zenrows/v1/scrapeUrlWithCssSelectors"
}
```

#### Output



Type: STRING
