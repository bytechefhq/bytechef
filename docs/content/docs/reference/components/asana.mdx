### Create Project
Name: createProject

`Creates a new project in a workspace or team.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| data | Data | OBJECT <details> <summary> Properties </summary> &#123;STRING\(workspace), STRING\(name), STRING\(notes), STRING\(team)&#125; </details> |  | false |

#### Example JSON Structure
```json
{
  "label" : "Create Project",
  "name" : "createProject",
  "parameters" : {
    "data" : {
      "workspace" : "",
      "name" : "",
      "notes" : "",
      "team" : ""
    }
  },
  "type" : "asana/v1/createProject"
}
```
