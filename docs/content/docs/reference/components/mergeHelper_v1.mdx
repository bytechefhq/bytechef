### Combine

|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| combineBy | Combine By | STRING <details> <summary> Options </summary> <span title="Merge items that have the same values for the specified key.">matchingFields</span>, <span title="Combine items based on their order in each input list.">position</span>, <span title="Merge every item from one input with every item from the others.">allPossibleCombinations</span> </details> | Determines how input items are combined. | true |
