## Actions


### Convert to Image
Name: convertToImage

`Converts pdf to image.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| file | PDF File | FILE_ENTRY | The PDF file which will be converted to image. | true |
| filename | Image Name | STRING | Name of the image. Every image will have index of the corresponding page in its name. | true |

#### Example JSON Structure
```json
{
  "label" : "Convert to Image",
  "name" : "convertToImage",
  "parameters" : {
    "file" : {
      "extension" : "",
      "mimeType" : "",
      "name" : "",
      "url" : ""
    },
    "filename" : ""
  },
  "type" : "pdfHelper/v1/convertToImage"
}
```

#### Output



Type: ARRAY


Items Type: FILE_ENTRY






#### Output Example
```json
[ {
  "extension" : "",
  "mimeType" : "",
  "name" : "",
  "url" : ""
} ]
```

### Image to PDF
Name: imageToPdf

`Converts image to PDF.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| images | null | ARRAY <details> <summary> Items </summary> [FILE_ENTRY\($image)] </details> | List of images that will be converted to one PDF. | true |
| filename | Filename | STRING | The name of the PDF file. | true |

#### Example JSON Structure
```json
{
  "label" : "Image to PDF",
  "name" : "imageToPdf",
  "parameters" : {
    "images" : [ {
      "extension" : "",
      "mimeType" : "",
      "name" : "",
      "url" : ""

  "type" : "pdfHelper/v1/imageToPdf"
```
