## Actions


### Scan
Name: scan

`Submit a URL to be scanned and control options for how the scan should be performed.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| url | URL | STRING | The URL to be scanned. | true |
| visibility | Visibility | STRING <details> <summary> Options </summary> public, unlisted, private </details> | Intended visibility of the final scan result. | false |
| tags | Tags | ARRAY <details> <summary> Items </summary> [STRING] </details> | User-defined tags to annotate this scan. | false |

#### Example JSON Structure
```json
{
  "label" : "Scan",
  "name" : "scan",
  "parameters" : {
    "url" : "",
    "visibility" : "",
    "tags" : [ "" ]
  },
  "type" : "urlscan/v1/scan"
}
```

#### Output



Type: OBJECT


#### Properties

|     Name     |     Type     |     Description     |
|:------------:|:------------:|:-------------------:|
| uuid | STRING | UUID for scan result, also called $scanId. |
| country | STRING | Country for scanning. |
| visibility | STRING | Determined visibility for scan. |
| url | STRING | Determined URL being scanned. |




#### Output Example
```json
{
  "uuid" : "",
  "country" : "",
  "visibility" : "",
  "url" : ""
}
```

# Additional Instructions

## Connection Setup

1. Login to the dashboard at https://urlscan.io/user/login/.
2. Click on Settings & API.
3. Click on New API key.
4. Add description and click Create API key.
6. Copy the API key. Use these credentials to create a connection in ByteChef.
