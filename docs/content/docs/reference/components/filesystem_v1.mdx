## Actions


### Create Temp Directory
Name: createTempDir

`Creates a file in the temporary directory on the filesystem. Returns the created directory's full path.`

#### Example JSON Structure
```json
{
  "label" : "Create Temp Directory",
  "name" : "createTempDir",
  "type" : "filesystem/v1/createTempDir"
}
```

#### Output


___Sample Output:___

```/sample_tmp_dir```



Type: STRING

### List
Name: ls

`Lists the content of a directory for the given path.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| path | Path | STRING | The path of a directory. | true |
| recursive | Recursive | BOOLEAN <details> <summary> Options </summary> <span>true</span>, <span>false</span> </details> | Should the subdirectories be included? | false |

#### Example JSON Structure
```json
{
  "label" : "List",
  "name" : "ls",
  "parameters" : {
    "path" : "",
    "recursive" : false
  },
  "type" : "filesystem/v1/ls"
}
```

#### Output



Type: ARRAY


Items Type: OBJECT


#### Properties
|     Name     |     Type     |     Description     |
|:------------:|:------------:|:-------------------:|
| filename | STRING | Name of the file. |
| relativePath | STRING | Relative path of the file. |
| size | INTEGER | Size of the file. |





#### Output Example
```json
[ {
  "filename" : "",
  "relativePath" : "",
  "size" : 1
} ]
```
