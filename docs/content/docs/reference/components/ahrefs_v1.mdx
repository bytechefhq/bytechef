## Actions


### Get Metrics
Name: getMetrics

`Returns metrics from target.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| target | Target | STRING | The target of the search: a domain or a URL. | true |
| date | Date | STRING | A date to report metrics on in YYYY-MM-DD format. | true |
| volume_mode | Volume Mode | STRING <details> <summary> Options </summary> <span>monthly</span>, <span>average</span> </details> | The search volume calculation mode: monthly or average. It affects volume, traffic, and traffic value. | false |
| protocol | Protocol | STRING <details> <summary> Options </summary> <span>both</span>, <span>http</span>, <span>https</span> </details> | The protocol of your target | false |
| output | Output | STRING <details> <summary> Options </summary> <span>json</span>, <span>csv</span>, <span>xml</span>, <span>php</span> </details> | The output format. | false |
| mode | Mode | STRING <details> <summary> Options </summary> <span>exact</span>, <span>prefix</span>, <span>domain</span>, <span>subdomains</span> </details> | The search volume calculation mode: monthly or average. It affects volume, traffic, and traffic value. | false |

#### Example JSON Structure
```json
{
  "label" : "Get Metrics",
  "name" : "getMetrics",
  "parameters" : {
    "target" : "",
    "date" : "",
    "volume_mode" : "",
    "protocol" : "",
    "output" : "",
    "mode" : ""
  },
  "type" : "ahrefs/v1/getMetrics"
}
```

#### Output



Type: OBJECT


#### Properties

|     Name     |     Type     |     Description     |
|:------------:|:------------:|:-------------------:|
| metrics | OBJECT <details> <summary> Properties </summary> &#123;INTEGER\(org_keywords), INTEGER\(paid_keywords), INTEGER\(org_keywords_1_3), INTEGER\(org_traffic), INTEGER\(org_cost), INTEGER\(paid_traffic), INTEGER\(paid_cost), INTEGER\(paid_pages)&#125; </details> |  |




#### Output Example
```json
{
  "metrics" : {
    "org_keywords" : 1,
    "paid_keywords" : 1,
    "org_keywords_1_3" : 1,
    "org_traffic" : 1,
    "org_cost" : 1,
    "paid_traffic" : 1,
    "paid_cost" : 1,
    "paid_pages" : 1
  }
}
```

### Get Page Content
Name: getPageContent

`Returns the content of a page.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| target_url | Target URL | STRING | The URL of the page to retrieve content for. | true |
| project_id | Project ID | STRING | The unique identifier of the project. Only projects with verified ownership are supported. | true |
| select | Select | STRING <details> <summary> Options </summary> <span>crawl_datetime</span>, <span>page_text</span>, <span>raw_html</span>, <span>rendered_html</span> </details> | A comma-separated list of columns to return. | true |

#### Example JSON Structure
```json
{
  "label" : "Get Page Content",
  "name" : "getPageContent",
  "parameters" : {
    "target_url" : "",
    "project_id" : "",
    "select" : ""
  },
  "type" : "ahrefs/v1/getPageContent"
}
```

#### Output



Type: OBJECT


#### Properties

|     Name     |     Type     |     Description     |
|:------------:|:------------:|:-------------------:|
| page-content | OBJECT <details> <summary> Properties </summary> &#123;STRING\(crawl_datetime), STRING\(page_text), STRING\(raw_html), STRING\(rendered_html)&#125; </details> |  |




#### Output Example
```json
{
  "page-content" : {
    "crawl_datetime" : "",
```
