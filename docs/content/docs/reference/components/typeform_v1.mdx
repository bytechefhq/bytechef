## Actions


### Create Form
Name: createForm

`Creates a new form`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| title | Title | STRING | Title to use for the form. | true |
| type | Type | STRING <details> <summary> Options </summary> <span>quiz</span>, <span>classification</span>, <span>score</span>, <span>branching</span>, <span>classification_branching</span>, <span>score_branching</span> </details> | Form type for the typeform. | false |
| workspace | Workspace | STRING | Workspace where the form will be created. | false |

#### Example JSON Structure
```json
{
  "label" : "Create Form",
  "name" : "createForm",
  "parameters" : {
    "title" : "",
    "type" : "",
    "workspace" : ""
  },
  "type" : "typeform/v1/createForm"
}
```

#### Output



Type: OBJECT
