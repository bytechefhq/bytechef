## Actions


### Create Contact
Name: createContact

`Creates a new contact.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| email | Email Address | STRING | Email address of the contact. | true |
| first_name | First Name | STRING | First name of the contact. | false |
| last_name | Last Name | STRING | Last name of the contact. | false |
| lists | Lists | ARRAY <details> <summary> Items </summary> [INTEGER] </details> | Lists to which the new contact will be added. | false |

#### Example JSON Structure
```json
{
  "label" : "Create Contact",
  "name" : "createContact",
  "parameters" : {
    "email" : "",
    "first_name" : "",
    "last_name" : "",
    "lists" : [ 1 ]
  },
  "type" : "sendfox/v1/createContact"
}
```

# Additional Instructions

## Connection Setup

### Find OAuth Client ID and Client Secret

1. Navigate to your dashboard.
2. Click this icon.
3. Click on **Settings**.
4. Click on **API**.
5. Click on **Create New Client**.
6. Enter OAuth client name.
7. Enter a redirect URI, e.g., http://127.0.0.1:5173/callback, https://app.bytechef.io/callback. Click on **Create**.
8. Here you can see your credentials.
