### Create Comment
Name: createComment

`Create a comment in a folder or in a task.`

|      Name       |      Label     |     Type     |     Description     | Required |

| parent | Parent | STRING <details> <summary> Options </summary> folders, tasks </details> | `Choose whether the comment will be added to a task or to a folder.` | true |
| parentId | Parent ID | STRING <details> <summary> Depends On </summary> parent </details> | `ID of the parent folder or the parent task.` | true |
| text | Text | STRING | `Comment text.` | true |
| plainText | Plain Text | BOOLEAN <details> <summary> Options </summary> true, false </details> | `Whether the comment will be treated as plain text or as HTML.` | false |

```json
}
```
