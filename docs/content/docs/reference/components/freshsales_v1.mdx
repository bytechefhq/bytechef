## Actions


### Create Account
Name: createAccount

`Creates a new account.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| name | Name | STRING | Name of the account. | true |
| website | Website | STRING | Website of the account. | false |
| phone | Phone | STRING | Phone number of the account. | false |

#### Example JSON Structure
```json
{
  "label" : "Create Account",
  "name" : "createAccount",
  "parameters" : {
    "name" : "",
    "website" : "",
    "phone" : ""
  },
  "type" : "freshsales/v1/createAccount"
}
```

#### Output



Type: OBJECT


#### Properties

|     Name     |     Type     |     Description     |
|:------------:|:------------:|:-------------------:|
| sales_account | OBJECT <details> <summary> Properties </summary> &#123;NUMBER\(id), STRING\(name), STRING\(website), STRING\(phone)&#125; </details> |  |




#### Output Example
```json
{
  "sales_account" : {
    "id" : 0.0,
    "name" : "",
    "website" : "",
    "phone" : ""
  }
}
```
