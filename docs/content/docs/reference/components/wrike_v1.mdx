## Connection Setup

### Find REST API Key

1. Navigate to your dashboard.
2. Click your account icon.
3. Click on **Settings**.
4. Click on **Profile**.
5. Scroll down and click on **Open Apps & Integrations**.
6. Click on **+App**.
7. Enter name of your application and below you can see your OAuth credentials.
8. Enter a redirect URI, e.g., http://127.0.0.1:5173/callback, https://app.bytechef.io/callback.
9. Click on **Save**.

### Create Comment
Name: createComment

`Create a comment in a folder or in a task.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| parent | Parent | STRING <details> <summary> Options </summary> <span>folders</span>, <span>tasks</span> </details> | Choose whether the comment will be added to a task or to a folder. | true |
| parentId | Parent ID | STRING <details> <summary> Depends On </summary> parent </details> | ID of the parent folder or the parent task. | true |
| text | Text | STRING | Comment text. | true |
| plainText | Plain Text | BOOLEAN <details> <summary> Options </summary> <span>true</span>, <span>false</span> </details> | Whether the comment will be treated as plain text or as HTML. | false |

```json
}
```
