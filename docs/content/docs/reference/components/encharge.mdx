## Actions


### Create Email Template
Name: createEmail

`Create email template`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| name | Name | STRING | `Name of the email template` | true |
| subject | Subject | STRING | `Subject of the email` | true |
| fromEmail | From Email | STRING | `From address to send the email from` | true |
| replyEmail | Reply Email | STRING | `Address that recipients will reply to by default.` | false |

#### Example JSON Structure
```json
{
  "label" : "Create Email Template",
  "name" : "createEmail",
  "parameters" : {
    "name" : "",
    "subject" : "",
    "fromEmail" : "",
    "replyEmail" : ""
  },
  "type" : "encharge/v1/createEmail"
}
```

#### Output



Type: OBJECT


#### Properties

|     Name     |     Type     |     Description     |
|:------------:|:------------:|:-------------------:|
| email | OBJECT <details> <summary> Properties </summary> &#123;INTEGER\(id), STRING\(name), STRING\(subject), STRING\(fromEmail), STRING\(replyEmail)&#125; </details> |  |




#### Output Example
```json
{
  "email" : {
    "id" : 1,
    "name" : "",
    "subject" : "",
    "fromEmail" : "",
    "replyEmail" : ""
  }
}
```
