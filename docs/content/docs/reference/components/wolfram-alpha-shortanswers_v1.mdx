## Connection Setup

### Find App ID

1. Navigate to your dashboard.
2. Click this icon.
3. Click on **Developer (API)**.
4. Click on **Get an App ID**.
5. Enter name, description and API. Select **DullResults API** or **Short Answers API** depending on which API you want to use.
6. Click on **Submit**.
