### Create Coupon
Name: createCoupon

`Create a new coupon.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| code | Code | STRING | Coupon code. | true |
| amount | Amount | STRING | The amount of discount. Should always be numeric, even if setting a percentage. | true |
| discount_type | Discount Type | STRING <details> <summary> Options </summary> <span>percent</span>, <span>fixed_cart</span>, <span>fixed_product</span> </details> | Determines the type of discount that will be applied. | true |
| description | Description | STRING | Coupon description. | false |
| date_expires | Date Expires | DATE_TIME | The date the coupon expires, in the site's timezone. | false |
| individual_use | Individual Use | BOOLEAN <details> <summary> Options </summary> <span>true</span>, <span>false</span> </details> | If true, the coupon can only be used individually. Other applied coupons will be removed from the cart. | false |
| product_ids | Product Ids | ARRAY <details> <summary> Items </summary> [INTEGER] </details> | List of product IDs the coupon can be used on. | false |
| exclude_sale_items | Exclude Sale Items | BOOLEAN <details> <summary> Options </summary> <span>true</span>, <span>false</span> </details> | If true, this coupon will not be applied to items that have sale prices. | false |
| minimum_amount | Minimum Amount | STRING | Minimum order amount that needs to be in the cart before coupon applies. | false |
| maximum_amount | Maximum Amount | STRING | Maximum order amount allowed when using the coupon. | false |

#### Example JSON Structure
