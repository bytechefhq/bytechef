## Actions


### Add Messages
Name: addMessages

`Adds messages to the vector store chat memory for a conversation.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| conversationId | Conversation ID | STRING | The unique identifier for the conversation. | true |
| messages | Messages | ARRAY <details> <summary> Items </summary> [&#123;STRING\(role), STRING\(content)&#125;] </details> | The messages to add to the conversation. | true |

#### Example JSON Structure
```json
{
  "label" : "Add Messages",
  "name" : "addMessages",
  "parameters" : {
    "conversationId" : "",
    "messages" : [ {
      "role" : "",
      "content" : ""
    } ]
  },
  "type" : "vectorStoreChatMemory/v1/addMessages"
}
```

#### Output



Type: OBJECT


#### Properties

|     Name     |     Type     |     Description     |
|:------------:|:------------:|:-------------------:|
| conversationId | STRING |  |
| messageCount | INTEGER |  |




#### Output Example
```json
{
  "conversationId" : "",
  "messageCount" : 1
}
```
