## Actions


### Raw Query
Name: rawQuery

`Run a raw query to a GraphQL endpoint.`

#### Properties

|      Name       |      Label     |     Type     |     Description     | Required |
|:---------------:|:--------------:|:------------:|:-------------------:|:--------:|
| graphQlEndpoint | GraphQL Endpoint | STRING | The endpoint where GraphQL requests will be sent. | true |
| headers | Headers | OBJECT <details> <summary> Properties </summary> &#123;&#125; </details> | Headers that will be sent in the HTTP request alongside the GraphQL query. | false |
| variables | Variables | OBJECT <details> <summary> Properties </summary> &#123;&#125; </details> | Variables that will be sent with the GraphQL query. | false |
| query | Query | STRING | Query that will be sent to the GraphQL endpoint. | false |
